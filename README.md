# 基于新词发现的短信分类前端系统

这是一个自包含的浏览器端演示系统，用于展示“基于新词发现的短信分类方法”的系统界面和核心处理闭环。

## 打开方式

直接用浏览器打开：

```text
/Users/chendile/Documents/知识产权/专利/专利-新词发现/基于新词发现的短信分类前端系统/index.html
```

## 已实现界面

- 待分类短信输入与样例短信切换
- 分类总览：短信类别、置信度、新词检测、变异检测、风险提示
- 多任务联合输出：短信分类、新词检测、变异词检测
- 多模态融合可视化：文本、数值、符号、链接四模态权重
- 处理链路：对应专利 S1-S8 的完整流程
- 新词发现：注意力、PMI、边界熵、标签关联值、可信关联值
- 动态词典：主词典、观察区、晋升与衰减说明
- 决策解释：关键词元、特征命中、新词使用、词表注入路径

## 说明

该页面不依赖后端服务，使用前端规则模拟现有 Python 原型中的处理链路，便于专利系统演示和界面验收。真实模型接入时，可将 `app.js` 中的 `analyze()` 替换为后端接口调用，并保持当前渲染结构不变。
