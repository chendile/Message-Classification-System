# 基于新词发现的短信分类前端系统

这是一个自包含的浏览器端演示系统，用于展示“基于新词发现的短信分类方法”的系统界面和核心处理闭环。

## 打开方式

推荐启动本地后端，以支持人工修正分类落盘：

```bash
node server.js
```

然后访问：

```text
http://localhost:3000
```

也可以直接用浏览器打开 `index.html` 查看静态演示，但人工修正只能在当前页面临时生效。

## 已实现界面

- 待分类短信输入与样例短信切换
- 分类总览：短信类别、置信度、新词检测、变异检测、风险提示
- 自动标注结合人工标注：样本分类结果支持人工修正并写入本地后端
- 新词发现：注意力、PMI、边界熵、标签关联值、可信关联值
- 动态词典：主词典、观察区、晋升与衰减说明
- 迭代图谱：展示新词进入观察区、晋升主词典和衰减保留过程

## 说明

页面使用前端规则模拟短信分类、新词发现和动态词典闭环。`server.js` 提供无依赖 Node 后端，用于保存人工修正分类，数据写入 `data/corrections.json`。真实模型接入时，可将 `app.js` 中的 `analyze()` 替换为后端接口调用，并保留当前渲染结构。
